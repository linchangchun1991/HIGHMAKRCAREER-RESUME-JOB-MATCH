方法一：通过网页直接部署（最简单！推荐！）

┌─────────────────────────────────────────────────────┐
│ 第1步：打开网站                                      │
└─────────────────────────────────────────────────────┘

1. 打开你的浏览器（Safari、Chrome 都可以）
2. 在地址栏输入：vercel.com
3. 按回车键

┌─────────────────────────────────────────────────────┐
│ 第2步：注册/登录                                     │
└─────────────────────────────────────────────────────┘

1. 点击右上角的 "Sign Up"（注册）按钮
2. 选择用 GitHub 登录（如果没有 GitHub 账号，先注册一个）
   - 访问 github.com 注册（免费，1分钟搞定）
   - 然后再回到 vercel.com 用 GitHub 登录

┌─────────────────────────────────────────────────────┐
│ 第3步：创建项目                                      │
└─────────────────────────────────────────────────────┘

1. 登录后，点击右上角绿色的 "Add New..." 按钮
2. 选择 "Project"

┌─────────────────────────────────────────────────────┐
│ 第4步：上传文件（最简单的方法）                      │
└─────────────────────────────────────────────────────┘

方法A：直接拖拽上传（最简单！）

1. 在 Vercel 的导入页面，找到 "Browse" 或 "Upload" 按钮
2. 打开你的 Finder（文件管理器）
3. 找到这个文件夹：/Users/changchun/Desktop/job_scraper
4. 把 index.html 文件拖拽到 Vercel 页面上
5. 点击 "Deploy" 按钮

方法B：通过 GitHub（稍微复杂但更专业）

如果你想要自动更新功能，可以先把代码放到 GitHub：

1. 访问 github.com 并登录
2. 点击右上角 "+" 号，选择 "New repository"
3. 仓库名填：resume-analysis（随便起名）
4. 选择 "Public"
5. 点击 "Create repository"
6. 然后告诉我，我帮你把代码推送到 GitHub

┌─────────────────────────────────────────────────────┐
│ 第5步：配置（很简单）                                │
└─────────────────────────────────────────────────────┘

1. Framework Preset: 选择 "Other"
2. 其他选项都留空，不用填
3. 点击绿色的 "Deploy" 按钮

┌─────────────────────────────────────────────────────┐
│ 第6步：等待完成                                      │
└─────────────────────────────────────────────────────┘

1. 等待 1-2 分钟
2. 看到 "Success" 或 "Ready" 就完成了！
3. 你会得到一个链接，类似：
   https://your-project.vercel.app
4. 点击这个链接就能访问你的网站了！
