# 招聘岗位自动化抓取项目

## 项目概述

这是一个基于 Playwright 和 Pandas 的自动化招聘岗位抓取脚本，主要用于为"就业困难/岗位稀缺"的学生群体抓取特定的岗位信息。

## 项目文件结构

```
.
├── job_scraper_main.py        # 主爬虫脚本（核心文件）
├── job_search_configs.py      # 搜索配置列表（43个配置项）
├── requirements.txt           # Python依赖包列表
├── INSTALL.md                 # 详细安装指南
├── job_scraper_README.md      # 功能说明文档
└── 项目说明.md                # 本文件
```

## 核心功能

### 1. 多网站支持
- ✅ **实习僧 (shixiseng.com)** - 优先支持，适合25/26届和实习岗位
- ✅ **前程无忧 (51job.com)** - 支持社招和全职岗位

### 2. 智能搜索
- 支持43个预配置的搜索条件
- 自动展开模糊地区（如"非偏远地区"→28个城市）
- 支持多关键词、多城市组合搜索
- 自动处理届数要求（2024/2025/2026届）

### 3. 数据提取
严格按照要求的9个字段提取：
- 公司名称
- 公司类型
- 工作地点
- 招聘类型
- 招聘对象
- 岗位(大都不限专业)
- 更新时间
- 投递截止
- 相关链接

### 4. 智能过滤
- **央国企过滤**：检查公司名称是否包含"国有"、"中国"、"集团"等关键词
- **大厂过滤**：检查是否在预定义的30+互联网大厂列表中
- **四大过滤**：检查是否包含"普华永道"、"德勤"、"安永"、"毕马威"

### 5. 数据去重
- 基于URL自动去重，避免重复数据

### 6. Excel导出
- 自动生成格式化的Excel文件
- 美化表头样式、列宽和行高
- 文件命名：`job_hunting_results_YYYY-MM-DD.xlsx`

## 配置说明

### 配置列表统计

- **总配置数**：43个
- **届数分布**：
  - 2026届：24个（最多）
  - 2025届：10个
  - 2024届：4个
  - 2023届：1个
- **招聘类型**：
  - 校招：37个
  - 社招/校招：4个
  - 社招：2个
- **总关键词数**：90个
- **涉及城市数**：49个

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
playwright install chromium
```

### 2. 运行脚本

```bash
python job_scraper_main.py
```

### 3. 查看结果

运行完成后，在当前目录查找 `job_hunting_results_YYYY-MM-DD.xlsx` 文件。

## 注意事项

⚠️ **重要提示**：

1. **遵守网站规则**：请遵守目标网站的robots.txt和使用条款
2. **合理使用**：不要过于频繁运行，避免给服务器造成压力
3. **数据用途**：抓取的数据仅供个人使用，请勿用于商业用途
4. **网站变化**：网站结构可能会变化，需要定期更新选择器
5. **网络环境**：确保网络连接正常，能够访问目标网站

## 故障排除

### 常见问题

1. **找不到职位数据**
   - 检查网络连接
   - 手动访问网站确认结构
   - 查看控制台错误信息

2. **被封IP**
   - 增加休眠时间
   - 减少配置数量
   - 使用代理IP

3. **浏览器启动失败**
   - 运行 `playwright install chromium`
   - 检查Python版本

详细故障排除请参考 `INSTALL.md`
