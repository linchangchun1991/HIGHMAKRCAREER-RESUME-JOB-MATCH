# 快速开始指南

## 第一步：安装依赖

在终端中进入项目目录，运行：

```bash
cd simple-job-match
npm install
```

## 第二步：运行项目

```bash
npm run dev
```

然后在浏览器中打开：http://localhost:3000

## 第三步：测试

1. 在页面的文本框中粘贴你的简历内容
2. 点击"立即匹配"按钮
3. 等待 AI 分析（几秒钟）
4. 查看匹配结果：推荐岗位、匹配度、推荐理由和改进建议

## 如何添加新岗位？

打开 `lib/data.ts` 文件，在 `JOBS` 数组中添加新岗位：

```typescript
{
  id: "6",  // 新的唯一ID
  title: "新岗位名称",
  company: "公司名称",
  salary: "薪资范围",
  description: "职责和要求描述"
}
```

保存后，新岗位会自动加入匹配库！

## 部署到 Vercel

1. **推送到 GitHub**
   - 在 Cursor 左侧点击 "Source Control"
   - 点击 "Publish to GitHub"
   - 选择仓库名称并发布

2. **在 Vercel 部署**
   - 访问 https://vercel.com 并注册/登录
   - 点击 "Add New Project"
   - 选择你刚才推送的 GitHub 仓库
   - **重要**：在 "Environment Variables" 中添加：
     - `ALI_API_KEY` = `sk-668c28bae516493d9ea8a3662118ec98`
     - `ALI_BASE_URL` = `https://dashscope.aliyuncs.com/compatible-mode/v1`
   - 点击 "Deploy"

3. **完成！**
   - 几分钟后，你的网站就上线了！
   - Vercel 会给你一个免费的域名（如：your-project.vercel.app）

## 常见问题

**Q: 匹配失败怎么办？**
A: 检查 `.env.local` 文件中的 API Key 是否正确，确保网络连接正常。

**Q: 可以添加更多岗位吗？**
A: 可以！直接在 `lib/data.ts` 中添加即可，没有数量限制。

**Q: 需要数据库吗？**
A: 不需要！所有岗位数据都存储在代码文件中，简单直接。
