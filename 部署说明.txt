📋 方法一：通过 Vercel 网页部署（最简单，推荐）

1. 打开浏览器，访问：https://vercel.com

2. 使用 GitHub/Google 账号登录（如果没有账号，先注册，完全免费）

3. 点击右上角 "Add New..." → "Project"

4. 选择 "Import Git Repository"
   - 如果项目已经在 GitHub：
     * 选择你的仓库
     * 点击 "Import"
   - 如果项目不在 GitHub：
     * 先在 GitHub 创建新仓库
     * 然后运行以下命令推送代码：
     
       cd /Users/changchun/Desktop/job_scraper
       git remote add origin https://github.com/你的用户名/仓库名.git
       git push -u origin main
     
     * 然后在 Vercel 中选择这个仓库

5. 配置项目：
   - Framework Preset: 选择 "Other"
   - 其他设置保持默认
   - 点击 "Deploy"

6. 等待 1-2 分钟，部署完成后你会得到一个链接：
   例如：https://your-project.vercel.app

7. 完成！这个链接可以分享给任何人使用！

═══════════════════════════════════════════════════════

📋 方法二：通过命令行部署（需要先安装 Node.js）

1. 安装 Node.js（如果还没有）：
   访问：https://nodejs.org 下载安装

2. 安装 Vercel CLI：
   npm install -g vercel

3. 登录 Vercel：
   vercel login

4. 部署：
   cd /Users/changchun/Desktop/job_scraper
   vercel --prod
