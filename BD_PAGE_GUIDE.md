# 企拓端页面使用指南

## ✨ 新功能特性

### 1. 岗位数据库管理
- ✅ 使用 React Context 管理岗位状态
- ✅ 支持添加、更新、删除岗位
- ✅ 数据持久化（当前为内存存储，可扩展为数据库）

### 2. AI 智能解析 JD
- ✅ 输入任意格式的招聘文本
- ✅ 一键解析，自动填充：
  - 岗位名称
  - 薪资范围
  - 岗位描述
  - 最低学历要求
  - 岗位要求（技能和经验）

### 3. 精美的 UI 设计
- ✅ Linear/Airbnb 风格的清爽设计
- ✅ 侧边抽屉（Drawer）替代传统弹窗
- ✅ 卡片式岗位列表展示
- ✅ 流畅的动画效果
- ✅ 响应式设计

## 🎨 设计特点

### 视觉风格
- **极简主义**：大量留白，清晰的层次结构
- **卡片设计**：每个岗位以独立卡片展示
- **图标系统**：使用 Lucide React 图标增强可读性
- **悬停效果**：鼠标悬停显示操作按钮

### 交互体验
- **侧边抽屉**：从右侧滑出，不遮挡主要内容
- **加载状态**：AI 解析时显示加载动画
- **即时反馈**：操作后立即显示 Toast 提示
- **空状态**：无数据时显示友好的引导界面

## 📋 使用流程

### 添加新岗位

1. **点击"新增岗位"按钮**
   - 右侧滑出抽屉面板

2. **方式一：AI 智能解析**
   - 在"AI 智能解析 JD"文本框中粘贴招聘文本
   - 点击"AI 识别"按钮
   - 系统自动填充岗位信息

3. **方式二：手动填写**
   - 直接填写各个字段
   - 可以结合 AI 解析结果进行修改

4. **完善信息**
   - 检查并修改自动填充的内容
   - 添加或删除岗位要求
   - 填写公司名称和地点

5. **保存岗位**
   - 点击"添加岗位"按钮
   - 岗位自动添加到列表顶部

### AI 解析示例

**输入文本：**
```
招聘高级前端工程师
薪资：25K-40K
地点：北京
要求：5年以上React开发经验，熟悉TypeScript、Next.js
学历：本科及以上
```

**解析结果：**
- 岗位名称：高级前端工程师
- 薪资范围：25K-40K
- 工作地点：北京（需手动填写）
- 最低学历：本科
- 岗位要求：React、TypeScript、Next.js、5年经验

## 🔧 技术实现

### Context 状态管理
```typescript
// contexts/jobs-context.tsx
- JobsProvider: 提供岗位数据上下文
- useJobs: Hook 用于访问岗位数据
- addJob: 添加新岗位
- updateJob: 更新岗位信息
- deleteJob: 删除岗位
- getJob: 获取单个岗位
```

### API 端点
```typescript
// app/api/parse-jd/route.ts
POST /api/parse-jd
Body: { jdText: string }
Response: {
  success: boolean,
  data: {
    title: string,
    salary: string,
    description: string,
    education: string,
    requirements: string[]
  }
}
```

### Drawer 组件
```typescript
// components/ui/drawer.tsx
- 基于 Radix UI Dialog 实现
- 支持左右两侧滑出
- 流畅的动画效果
- 响应式设计
```

## 📱 响应式支持

## 📝 注意事项

1. **API Key 配置**：确保 `.env.local` 中配置了阿里云 API Key
2. **数据持久化**：当前数据存储在内存中，刷新页面会重置
3. **文件大小**：AI 解析的文本建议不超过 5000 字
4. **解析准确性**：AI 解析结果可能需要人工校验和调整
