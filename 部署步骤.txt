   🚀 部署到 Vercel - 超简单步骤（5分钟）
═══════════════════════════════════════════════════════

✅ 你的文件已经准备好了！
   - index.html ✅
   - vercel.json ✅

现在只需要 3 步：

═══════════════════════════════════════════════════════
   步骤 1：打开 Vercel 网站
═══════════════════════════════════════════════════════

1. 打开浏览器（Safari、Chrome 都可以）
2. 在地址栏输入：vercel.com
3. 按回车

═══════════════════════════════════════════════════════
   步骤 2：登录账号
═══════════════════════════════════════════════════════

1. 点击右上角 "Sign Up" 或 "Log In"
2. 选择 "Continue with GitHub"（用 GitHub 登录）

   ⚠️ 如果你没有 GitHub 账号：
   - 先访问 github.com
   - 点击 "Sign up" 注册（免费，1分钟）
   - 然后再回到 vercel.com 登录

═══════════════════════════════════════════════════════
   步骤 3：上传并部署
═══════════════════════════════════════════════════════

1. 登录后，点击右上角 "Add New..." → "Project"

2. 在导入页面，选择 "Browse" 或直接拖拽：
   - 找到文件夹：桌面 → job_scraper
   - 选择文件：index.html
   - 拖拽到页面上

3. 配置项目：
   - Framework Preset: 选择 "Other"
   - 其他都留空，不用填

4. 点击绿色的 "Deploy" 按钮

5. 等待 1-2 分钟

6. 完成！你会得到一个链接，类似：
   https://your-project.vercel.app

═══════════════════════════════════════════════════════
   🎉 完成！
═══════════════════════════════════════════════════════

现在你的网站已经上线了！
- 可以分享链接给任何人
- 可以在任何设备上访问
- 完全免费
