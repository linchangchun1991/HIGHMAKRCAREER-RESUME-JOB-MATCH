# 简历批量下载 - 快速开始

## 第一步：安装依赖

```bash
cd /Users/changchun/Desktop/job_scraper
pip install playwright
playwright install chromium
```

## 第二步：运行脚本

```bash
python resume_downloader.py
```

## 第三步：操作流程

1. **浏览器会自动打开**，访问登录页面
2. **输入登录信息**：
   - 脚本会尝试自动填写，如果失败可以手动输入
   - 或者直接在浏览器中手动登录
3. **等待下载完成**：
   - 脚本会自动识别表格
   - 下载所有简历文件
   - 根据学员名称重命名

## 下载的文件位置

所有简历会保存在：
```
/Users/changchun/Desktop/job_scraper/resumes/
```

文件名格式：`{学员名称}.pdf` 或 `{学员名称}.docx`

## 常见问题

### Q: 找不到表格怎么办？
A: 脚本会自动保存页面HTML到 `resumes/page_debug_*.html`，可以查看页面结构

### Q: 下载失败怎么办？
A: 
- 确保已正确登录
- 检查网络连接
- 查看控制台错误信息

### Q: 文件名不对怎么办？
A: 脚本会根据"名气直推投递"列的内容重命名，如果该列为空，会使用默认名称

## 高级选项

如果需要修改下载目录或使用无头模式：

```python
# 编辑 resume_downloader.py，修改 main() 函数中的参数
downloader = ResumeDownloader(
    download_dir="/自定义/路径",  # 自定义下载目录
    headless=True  # 后台运行（不显示浏览器）
)
```
